# B 风格商业落地型沙盘 · v2

> 单页 3D 沙盘 · v2-tool 版本的「改进版」
> 设计哲学：**顶栏命令条 + 右侧统一仪表盘**，砍掉散点浮窗，3D 沙盘占屏更大

---

## 一、3 秒启动

```bash
cd v3-sandbox-b
python3 -m http.server 8001
```

浏览器打开 `http://localhost:8001/` 即可。

> 注意端口用 **8001**，避免与 v2-tool（8000）冲突，可以两个窗口对比看。

---

## 二、与 v1（v2-tool）的差异

| 元素 | v1（v2-tool 端口 8000） | v2（v3-sandbox-b 端口 8001）⭐ |
|---|---|---|
| **整体布局** | 5 个磨砂浮窗散落 | 顶栏 + 右侧统一仪表盘 |
| **控件位置** | 顶中竖叠 3 排 | 顶栏一行排齐 |
| **3D 沙盘** | 占屏约 60% | 占屏约 70% |
| **HERO 卡** | 4 KPI 平铺 | 深色英雄卡 + 大号金色评分 |
| **决策叙事** | ❌ 无 | ✅ 自动生成中文叙事文案 |
| **5 年现金流** | ❌ 无 | ✅ 迷你曲线 + IRR/NPV/回本年 |
| **健康检查** | ❌ 无 | ✅ 4 项行业警戒线 + 绿黄红卡 |
| **图表洞察** | ❌ 无 | ✅ 每张图下方有"为什么"洞察 |
| **铺位脉冲** | 蓝色光柱呼吸 | + 地面脉冲环 |

**核心理念**：v1 是"沙盘 + 控件"，v2 是"**决策仪表盘 + 沙盘**"——从"看数据"到"读懂数据"。

---

## 三、新增的 5 张洞察

每张图表下方都有一句**自动生成的中文判断**：

1. **五维雷达**：找出最强/最弱维度，提示是否均衡
2. **客群分层**：识别核心贡献客群，给业态运营建议
3. **U 曲线**：判断"蓝海/甜蜜点/超载"三种状态
4. **5Y 现金流**：根据 NPV 给三档结论（好/普通/亏）
5. **健康检查**：4 项指标对照行业警戒线（绿/黄/红）

## 四、文件结构

```
v3-sandbox-b/
├── index.html         单页 B 沙盘 v2（约 800 行）
├── README.md          本文件
├── data/              共享数据层（与 v2-tool 一致）
│   ├── spots.js
│   ├── categories.js
│   └── investorTiers.js
```
