# 商业落地型沙盘 · 天府国际金融中心

> 单页 3D 沙盘，B 套形态 · 苹果白 / 铂金 / 磨砂玻璃
> 公式组合：G② + E② + C③ + K② + X②
> 数据：30 个虚拟铺位 + 3 业态 + 12 月季节 + 4 时段

---

## 一、3 秒启动

```bash
cd v2-tool
python3 -m http.server 8000
```

浏览器打开 `http://localhost:8000/` 即可。

> 注意：HTML 用 ES 模块 + CDN，**必须用 HTTP 服务器**，不能直接 `file://` 双击。

## 四、核心公式

| 维度 | 套餐 B 选档 | 含义 |
|---|---|---|
| ① 地理 G | ② 进阶版（距离衰减 exp(-d/200) + 介数中心性） | 地段、地铁、楼层、动线 |
| ② 经济 E | ② 进阶（时段权重 1+τ×0.5） | 流量 × 消费力 × 时段集中度 |
| ③ 消费者 C | ③ 专业（4 类客群分漏斗） | 进店 → 转化 → 提袋 |
| ④ 竞争 K | ② U 形曲线（集聚 vs 超载） | "开第几家最赚"问题 |
| ⑤ 外部 X | ② 进阶（季节调整 1+season_m） | 政策 + 月度修正 |

完整公式见 `lib/calc.js`，每个函数都是纯函数可独立测试。

## 五、给 Claude / 同事接力的提示

### 改业态
编辑 `data/categories.js`，每个业态有这些字段：
- `AOV / r1 / r2 / r3` — 客单价、进店率、转化率、提袋率
- `netMargin / monthlyOpex / investBase / investPerM2` — 财务
- `segments` — 4 类客群占比和漏斗
- `season` — 12 个月季节修正
- `N_carry` — 商圈最大承载同业数（影响 U 曲线）

### 改铺位
编辑 `data/spots.js` 的 `SPOTS` 数组。30 个铺位每个都有坐标、面积、租金、五维原始指标。

### 改沙盘建筑
编辑 `data/spots.js` 的 `LANDMARKS / METRO` 数组。坐标全部用米，IFC 为原点。

### 改风格
`lib/styles.css` 顶部 `:root` 改 CSS 变量即可全局换色。

### 接真实数据
把 `data/*.js` 改成 `fetch()` API 即可对接后端。
