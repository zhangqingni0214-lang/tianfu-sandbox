## 三、五大变量 & 计算逻辑

完整文档见 **`五大变量计算逻辑.md`**，包含：

1. **地理性 G**（Geography）：地段、地铁距离、街道引导、楼层、动线
2. **经济动态 E**（Economic Dynamics）：人流 × 钱包 × 意愿
3. **消费者画像 C**（Consumer Profile）：进店/转化/提袋/客单/复购 漏斗
4. **竞争生态 K**（Competition Ecosystem）：集聚效应 vs 同业饱和（U 形）
5. **外部性 X**（Externality）：政策、舆情、季节、媒体

每个维度配 ①基础 / ②进阶 / ③专业 三档公式变体，可自由组合。

## 七、给同事 / 给 Claude 接力的提示

如果你的同事想：

- **改建筑布局** → 编辑 HTML 里的 `BUILDINGS` 数组，每条 `{x, z, w, d, h}` 单位都是米
- **改公式参数** → 顶部 `DATA` 常量，例如 `propensity_pi`（消费倾向）、`anchor_w`（锚店权重）
- **换品类** → B 套里的 `DATA.categories`，C 套里的 `DATA.category`
- **换商圈** → 把 `BUILDINGS / METRO / SOURCES` 全替换成新地块的坐标即可，公式不用动
- **接入真实数据** → 把 `DATA` 改成从后端 API 拉取的数据结构即可

把整个文件夹拖进 Claude Code，Claude 会自动读到 `CLAUDE.md`，立刻知道这是什么项目、如何继续迭代。
