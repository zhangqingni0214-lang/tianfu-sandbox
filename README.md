## ✨ v4 极简推演版核心功能（v2.0）

### 🎯 单地块推演
- 中央 1 个金色目标铺位（呼吸光晕 + 光柱），周边 4 类背景影响因素
- 4 类背景：🚇 地铁站 / 🏢 4栋写字楼 / 🏠 8栋住宅 / 🏬 银泰 in99
- 简化交互：仅旋转 + 缩放，不再有寻路、小人、二级沙盘等干扰

### 📐 右栏 5 步推演（核心创新）
1. **背景客流** — 4 类来源 × 距离衰减 × 频次 → 日均潜在路过
2. **业态吸引** — × flowMix × 进店率 → 日有效到店（每个业态的客源结构不同）
3. **漏斗转化** — × r2 × r3 × AOV → 日营收
4. **修正系数** — × 竞争 K · 同向 M · 季节
5. **年度财务** — 利润 / 投资 → ROI · 回收期 · 租金占比

### 🔄 业态对比模式
- 点击"+ 对比业态"叠加第二个业态，5 步全部并排展示
- 自动结论：「同一铺位，💄 美妆护肤 比 ☕ 精品咖啡 多赚 135 万/年」
- 关键差异提示：哪个业态的客源结构与本铺位更匹配

### 🏆 TOP 业态推荐
- 全 8 业态扫描，按年净利排序展示 #1~#5
- 一键切换查看任何业态的完整推演

### 💡 联动高亮
- 悬停右栏步骤 → 沙盘对应背景体高亮、客流线动画加速
- 悬停沙盘 chip → 对应步骤数据呼应

## ✨ v3 沙盘核心功能（v1.0 完整版）

### 🏙 外层沙盘（金融城 1km²）
- 30 个虚拟候选铺位（按区域分组：IFC / 双子塔 / in99 / 交子 / 住宅配套）
- 16 个 L2 业态（咖啡 / 茶饮 / 火锅 / 时装 / 美妆 / 健身 / 影院 ...）
- 纪念碑谷 Ida 风格白色小人，点击建筑会沿金色虚线绕路走过去（不穿模）
- 走路时镜头自动俯视，到达后缓动回原视角
- 滚轮缩放 / 拖拽旋转 / 双击空地重置视角

### 🏢 二级沙盘（建筑内部）
- 8 栋商业建筑可"进入"（IFC T1/T2 / 双子塔 / in99 / 交子 / 华商）
- Three.js 真 3D 楼层展示，每个铺位 = 立方体（顶面贴店铺名 + emoji）
- 按楼层分布业态（B1 美食街 / 1F 临街 / 2-3F 餐饮零售 / 写字楼 ...）
- ↑/↓ 切楼层 · ESC 退出

### 📊 评估仪表盘
- 5 维度评分（地理 G / 经济 E / 消费者 C / 竞争 K / 外部 X）
- 4 KPI（综合评分 / 年营收 / ROI 回收期 / 租金比）
- 客群分层（金融白领 / 居民 / 商旅 / 其他 4 类漏斗）
- U 形竞争曲线（同业数 vs 修正系数，标当前+最优）
- 5 年累计现金流 + IRR + NPV
- 投资健康检查（4 项行业警戒线）
- 📌 快照对比（最多 3 个评估并排）

### ⚙ 计算逻辑
五大变量计算逻辑详见 [`五大变量计算逻辑.md`](./五大变量计算逻辑.md)。
- G 地理：GIS 简化版（POI × 距离² + 街道介数）
- E 经济：时段权重 + Huff 引力
- C 消费者：4 客群分漏斗
- K 竞争：U 形曲线 + 定位差异化
- X 外部：政策 + 季节 + （蒙卡 in 套餐 C）

## 🛠 本地运行

```bash
git clone https://github.com/zhangqingni0214-lang/tianfu-sandbox.git
cd tianfu-sandbox
python3 -m http.server 8002
# 浏览器打开：
#   http://localhost:8002/商业落地型沙盘demo2/    （v2.0 极简推演版 · 推荐）
#   http://localhost:8002/v3-sandbox-b/         （v1.0 完整版）
```
